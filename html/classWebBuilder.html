Detailed Description

Builds webs from live ranges.

Live ranges belonging to the same variable are merged into the same web whenever they overlap or share common program points.

Builds webs from a set of live ranges.

Overlapping live ranges associated with the same variable are merged into a single web.

Parameters

Time Complexity: O(R² * P²)

where:
